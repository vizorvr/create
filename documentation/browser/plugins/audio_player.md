#Audio player

##Description
Play an audio stream. Playback loops.

##Inputs
###audio
The **audio stream** to play.

###play
Send **True** to start playback and **False** to stop.

###mute
Send **True** to mute playback and **False** to re-enable audio.

###volume
Set playback **volume**.

###time
Set playback **time**.
