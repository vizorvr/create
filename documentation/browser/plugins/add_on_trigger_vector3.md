#Add Vectors Conditionally

##Description
When receiving **True**, will output **A** + **B**, otherwise **A**.

##Inputs
###trigger
On **True**, this plugin will output **A** + **B**, otherwise **A**.

###A
Value to add **B** to.

###B
Value to add to **A**.

##Outputs
###value
if trigger is **True**, will output **A** + **B**, otherwise **A**.
