#Blend color

##Description
Color crossfader. Performs a linear blend between two colors (Color **A** and Color **B**).

##Inputs
###color A
First color **operand**.

###color B
Second color **operand**.

###mix
**0**: Emit only color **A**.
**1**: Emit only color **B**.

##Outputs
###color
Linear mix of colors **A** and **B**: A * (1 - mix) + B * mix
