#XYZ to Vector

##Description
Create a **vector** from individual **X**, **Y** and **Z** components.<br><br>Keyboard shortcut&nbsp;&nbsp;&nbsp;<kbd>7</kbd>

##Inputs
###x
The input X-component.

###y
The input Y-component.

###z
The input Z-component.

##Outputs
###vector
The resulting **vector**.
